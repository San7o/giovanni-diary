Programming

I wrote my first lines of code when I was about 13. From there I tried game development with Unity, then programming competitions and later CTFs. Through the competitions I was introduced to Linux, I learned about UNIX via "Advanced Programming in the Unix Environment" by talking to Leonardo Taccari.

After doing some web applications in university, my first intermediate project was a game engine written in C++ and opengl. The code became big enough that parts of It became their own repos: oak, viotecs and valfuzz. I wanted to understand C++ deeper, so I reimplemented the standard library.

During this time I was getting passionate about operating systems. I wrote a malware scanner using a linux kernel module and I began working on my own operating system. I started using Linux From Scratch and Emacs daily and I became a strong supporter of the free software movement. I got into cloud computing thought an internship where I wrote a kubernetes operator that uses eBPF programs.
